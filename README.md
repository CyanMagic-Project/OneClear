# OneClear .B - FFmpeg Re-linking Data

This repository provides the object codes and instructions for re-linking FFmpeg with OneClear .B, in compliance with the GNU Lesser General Public License (LGPL) v2.1.

## How to Re-link

1. Download the pre-compiled object files of OneClear .B from this repository.
2. Prepare your own compiled static library of FFmpeg (`libavcodec.a`, `libavformat.a`, etc.).
3. Run the linker command to replace the FFmpeg library and generate the modified executable.
